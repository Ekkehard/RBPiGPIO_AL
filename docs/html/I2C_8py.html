Detailed Description

Version
2.0.0
Purpose
This module provides an I2C bus abstraction layer for the Raspberry Pi General Purpose I/O (GPIO) functionality for all models of the regular Raspberry Pi 0 and up as well as the Raspberry Pi Pico. Using this module, code should run and use the common functionality of the GPIO shared by all Raspberry Pi architectures without modifications on all those architectures.

This code has been tested on a Raspberry Pi 0, 3, 4 and 5 and a Raspberry Pi Pico.
