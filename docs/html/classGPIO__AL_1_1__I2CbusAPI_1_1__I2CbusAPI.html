Parameters
sdaPinheader pin or GPIO line number for I2C data (default GPIO2 on Raspberry Pi and 8 on Raspberry Pi Pico). Ints are interpreted as header pin numbers, strings starting with GPIO as line numbers.
sclPinheader pin or GPIO line number for I2C clock (default GPIO3 on Raspberry Pi and 9 on Raspberry Pi Pico). Ints are interpreted as header pin numbers, strings starting with GPIO as line numbers.
modeone of I2Cbus.Mode.HARDWARE or I2Cbus.Mode.SOFTWARE
frequencyI2C frequency in Hz (default 75 kHz for Software mode and 100 kHz for hardware mode and Raspbberry Pi Pico in all modes). This parameter is ignored for Raspberry Pi in hardeware mode, where the frequency is always 100 kHz. Also accepts PObjects of Unit Hz.
attemptsnumber of read or write attempts before throwing an exception
usePECset True to use Packet Error Checking
Exceptions
GPIOExceptionin case of parameter error
